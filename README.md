# SC-Ore-Scanner
Mining signature identification assistant for Star Citizen

SC Ore Scanner reads mining signature numbers displayed on your screen and
identifies the detected ore, rarity and node count.

![SC Ore Scanner](screenshots/main-window.png)

## Download

[Download the latest version](https://github.com/Gaban666/SC-Ore-Scanner/releases/latest)

No installation is required. Extract the downloaded ZIP and run:

`SC Ore Scanner.exe`

Keep all included files and the `_internal` folder together.

## Features

- Identifies mining signatures displayed on the HUD
- Shows detected ore, rarity and node count
- Continuous screen scanning mode
- Voice Request mode with offline voice recognition
- Speaks scan results using an onboard-computer-style voice
- Editable ore price table
- Built-in update checking
- Private bug report and suggestion form
- Fully portable application

## Voice Request Mode

Prefer to keep your eyes on the game?

Select **Voice Request**, click **Start**, complete the microphone test and ask
the onboard mining assistant to scan the visible signature.

Say:

- `scan`
- `scan this`
- `scan this sh**`

The assistant will scan the visible signature and tell you what it found.

## Continuous Mode

Select **Continuous** and click **Start**.

The application scans the signature area every 500 ms. Spoken results can be
enabled or disabled using the **Voice is ON / Voice is OFF** button.

Continuous mode is experimental and may be unstable on some computers.

## Screen Support

Currently calibrated only for **16:9 screens at 4K resolution**.

To request support for another resolution, use the **Report issue / Suggestion**
button inside the application. Include:

- a full-screen screenshot with the signature clearly visible
- your exact screen resolution
- your screen ratio
